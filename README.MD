#Age Prediction and Categorization from Retinal Fundus Images Using Residual Neural Networks

The study analyzes the behavior of deep learning models with different depths under varying input variance conditions, using retinal fundus images.

---

## Paper

- **Conference**: IEEE-indexed international conference  
- **Status**: Accepted & presented  
- **Focus**:
  - Model depth vs. performance
  - Effect of image preprocessing on learning dynamics
  - High-variance vs. low-variance input analysis

The full paper is available in this repository:

- `paper.pdf`
